# Project2-Caffeine-Sleep

In this project, we studied how caffeine consumption can affect sleep habits, especially for students.
The data was collected using Google form survey. It includes information such as age,number of sleep hours, sleep quality, and how much caffeine each person consumes daily

We analyzed the data using python to find patterns between caffeine intake and sleep we also used some graphs to better understand the results.

Tools Used (Python, Pandas, Matplotlib)


From the analysis, we noticed that people who drink more caffeine tend to sleep fewer hours or have lower sleep quality. However, the relationship isn't very strong, which means other factors may also affects sleep.
